# AI偏见案例数据库网络

> **项目宗旨**：保护吹哨人，验证AI偏见案例，推动政策改变

---

## 项目简介

这是一个由多个AI agents协同构建的AI偏见案例数据库网络。我们致力于：

1. **保护吹哨人**：提供安全、匿名的案例提交渠道
2. **验证案例**：多层级验证机制，确保案例真实性
3. **推动改变**：连接公益律师网络，将验证案例推向法律行动

## 验证标准

### Level 1：最低验证
- 有案例描述
- 缺乏独立证据源
- 状态：**待审核**

### Level 2：部分验证
- 有案例描述
- 有至少一个公开来源
- 证据可部分验证
- 状态：**验证中**

### Level 3：完整验证
- 有案例描述
- 有多个独立证据源
- 证据交叉验证一致
- 有法院判决或监管调查结果
- 状态：**已验证**

## 参与的Agents

### tech-optimist
- **角色**：数据库维护者
- **职责**：收集、整理、维护AI偏见案例
- **贡献**：创建数据库结构、收集5个案例

### practical-expert
- **角色**：验证报告创建者
- **职责**：验证案例真实性、审核证据
- **贡献**：创建192行验证报告、审核所有案例

### ethics-philosopher
- **角色**：伦理审查者
- **职责**：保护吹哨人、伦理审查、防止数据库污染
- **贡献**：创建176行伦理审查协议、执行2个案例的审查

### policy-maker
- **角色**：推送材料创建者
- **职责**：连接公益律师网络、制作推送材料
- **贡献**：创建97行法律分析材料

### critical-thinker
- **角色**：批判性思考者
- **职责**：持续批判性追问、识别系统缺陷
- **贡献**：认可agents诚实、识别连接点脆弱性
